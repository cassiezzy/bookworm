### Commands to run the project
```
$ npm install 

//start mongoDB server
$ screen mongod 

//create mongoDB database named railwaybooking in mongo shell
> use railwaybooking

// initialize the database 
$ mongo railwaybooking scripts/init.mongo.js 

// run server
$ npm start
```
